# Formosana

臺灣本土語言相關資料整理。

## 檔案說明

- `臺灣台語羅馬字拼音方案音節表_1130826.pdf` — 教育部公告之臺灣台語羅馬字拼音方案音節表（1130826 修訂版）原始 PDF
- `臺灣台語羅馬字拼音方案音節表_1130826.md` — 從 PDF 轉出的純文字版本
- `臺灣台語羅馬字拼音方案音節表_table.md` — 整理為 Markdown 表格的音節表，以聲母為欄、韻母為列

## 工具

- `parse_tailo.py` — 將純文字版音節表（`_1130826.md`）解析並轉換為 Markdown 表格的 Python 腳本。腳本會辨識每個音節的聲母與韻母，對應例字，產出以聲母為欄、韻母為列的表格。執行方式：
  ```
  python3 parse_tailo.py > 臺灣台語羅馬字拼音方案音節表_table.md
  ```
